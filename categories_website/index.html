Defect Categories

  • Category A — IV current too high: IV curve has current values above threshold.
  • Category B — Input noise too high: Input-noise curves have high outliers, bumps, wiggles, or values exceeding 2000.
  • Category C — Input noise too low: Input-noise values are unexpectedly low, possibly indicating dead or disconnected channels.
  • Category D(i) — Incomplete IV dataset: IV script ran, but only part of the expected IV data was generated.
  • Category D(ii) — Incomplete input-noise dataset: Input-noise script ran, but only part of the expected input-noise data was generated.
  • Category E(i) — IV data unavailable: IV data could not be found or processed.
  • Category E(ii) — Input-noise data unavailable: Input-noise data could not be found or processed.
